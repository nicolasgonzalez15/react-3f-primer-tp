# REACT JS/VITE 3F - Primer trabajo práctivo
Proyecto de React JS utilizando VITE como gestor de dependencias / ejecución de proyecto

## Propósito del proyecto
Construir una aplicación que permita:
* Recuperar items de un archivo js (colección)
* Mostrar datos de cada item en formato de card (Título/Categoría/Fecha/Favorito)
* Filtrar items según título o categoría
* Asignar localmente si es Favorita o no
* Manejar estilos dinámicos en CSS

## Tecnologías usadas
* React JS
* Javascript
* VITE
